# iaçá - Plataforma de Comercialização de Caroços de Açaí

Aplicativo React Native com Expo para conectar vendedores (feirantes) e empresas interessadas na compra de caroços de açaí.

## Tecnologias Utilizadas

- React Native com Expo
- React Navigation
- TypeScript
- Styled Components (ou StyleSheet do RN)
- Fast Refresh / Live Reload

##  Pré-requisitos

- [Node.js](https://nodejs.org/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- [Git](https://git-scm.com/)
- Um emulador Android/iOS ou o app **Expo Go** no seu celular

## 🛠️ Instalação

1. **Clone o repositório:**
Claro! Aqui está a versão corrigida, limpa e formatada do trecho que você tentou colar no `README.md`, sem repetições e com markdown apropriado:

---

###  Instalação

```bash
git clone https://github.com/caikymorais/iaca-app.git
cd iaca-app
```

**Instale as dependências:**

```bash
npm install
```

**Instale o Expo CLI (caso ainda não tenha):**

```bash
npm install -g expo-cli
```

---

###  Rodando o Projeto

####  Via celular (Expo Go):

```bash
npm start
```

* Escaneie o QR code com o app **Expo Go**

####  Via emulador Android:

```bash
npm run android
```

####  Via navegador (modo web):

```bash
npm run web
```
